# Riepilogo Versione 0.0.2

**Datazione:** 15 Aprile 2026.
**Cambiamenti Principali (Database & Core Backend):**
- **Inizializzazione db.rs:** Implementazione della vera essenza "Host-Side-Heavy". Il database SQLite locale gestisce ora fisicamente i metadati radio al riparo dal Javascript debole.
- **Risoluzione multios:** Integrata l'API protetta `tauri::app::path::app_data_dir()` in Rust in modo da tracciare la corretta directory di Windows (`AppData`), Mac (`Application Support`) e Linux (`.config`), mantenendo l'esperienza standard che ci si aspetta da un'app locale professionale e invisibile.
- **IPC Esibito:** Generato il primo command testabile `get_tracks()`, collegato alla tabella principale.

**Stato di Completamento Architetturale:**
Da questo momento in avanti, l'app può parlare con logiche esterne complesse scritte in Rust passando tutto via invoke ("`get_tracks`"). Costituisce la fine della configurazione infrastrutturale fondamentale (Fase 2 chiusa).
