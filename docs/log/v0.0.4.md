# Riepilogo Versione 0.0.4

**Datazione:** 15 Aprile 2026.
**Cambiamenti Principali (Interfaccia e UX):**

- **Nuovo Layout Modulare:** Abbandonato lo scaffolding iniziale di Vite in favore di un layout scuro a 3 pannelli tipico delle automazioni radio (Tailwind CSS 4).
- **Libreria Multimediale e Drag&Drop Nativi:** Introdotto il `LeftPanel` per listare le tracce dal DB SQLite. Abilitato il Dragging dei brani con l'ausilio delle native HTML5 DataTransfer API per un'estrema efficienza.
- **Mixer Coda Eventi (Droppable Target):** Sviluppata l'area `CenterPanel` adibita alla messa in onda. Il rilascio dei brani (Drop) li aggancia alla coda in memoria. L'attivazione con il tasto `PLAY NEXT` consuma l'elenco riproducendoli tramite i bind IPC (`play_audio`) di Rust senza bloccare mai l'UI.
- **Controlli Radio DSP:** Nel `RightPanel` implementato il vero tasto di `DJ MIC LIVE` per comandare al motore audio Rust di abbassare il db della libreria in background. Aggiunto V-Meter visuale logico L/R.
- **Mock Mode:** Aggiunto un comando `insert_mock_tracks` nel file `lib.rs` per popolare agevolmente la lista qualora SQLite sia vuoto, accessibile da pulsante grafico quando la libreria restituisce zero risultati.

**Stato di Completamento Architetturale:**
*Fase 4 Completata:* UI e sistema d'interazione utente interamente cablati al reattore Rust sottostante in via asincrona. Pronto per scheduler!
